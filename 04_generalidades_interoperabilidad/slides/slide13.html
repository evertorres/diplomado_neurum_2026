Categorías de Estándares en Salud

Los estándares pueden agruparse fundamentalmente en 3 pilares o "cajas de herramientas".

1. Mensajería / Intercambio

Proveen el formato general, la estructura para empaquetar datos y las reglas del protocolo para mover el mensaje entre sistemas.

Ejemplos: HL7 v2, HL7 v3, FHIR.

2. Terminología y Vocabulario

Constituyen la base terminológica (diccionarios y códigos numéricos rigurosos) para diagnosticar de forma unificada.

Ejemplos: SNOMED-CT, CIE-10, LOINC.

3. Documentos y Arquitectura

Definen la arquitectura de alto nivel de lo que constituye un documento clínico vital (qué secciones debe tener un resumen de egreso).

Ejemplos: CDA (Clinical Document Architecture), CCR.
