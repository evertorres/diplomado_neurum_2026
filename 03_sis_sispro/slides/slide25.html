OpenEHR Modelo de 2 capas

Modelo de 2 capas

Modelo de Referencia
(Información)
  • Modelo de información: Orientado a objetos
  • Estable:
    • Elementos reusables por el modelo de arquetipos
    • No se modifican en el tiempo
  • Fechas, textos, listas, etc.
  • Unidades, cantidades, descripciones, etc.
  • Rol, entidad, observación, acción, etc.
Modelo de Arquetipos
(Conocimiento)
  • Modelo de conocimiento: Representa conceptos
  • Dinámicos: Evoluciona según los conceptos del dominio
  • Presión arterial, APGAR, temperatura, laboratorios, medicación
  • Médico, paciente.
