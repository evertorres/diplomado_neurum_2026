Modelos de datos clínicos

¿Qué determina un modelo de datos?

Un modelo de datos determina:

  • Elementos a ser guardados: Definición de qué información es relevante.
  • Relaciones entre datos: Cómo se conectan las diferentes entidades.
Aspectos técnicos:
  • Tipos de datos
  • Rangos permitidos
  • Datos requeridos u opcionales
