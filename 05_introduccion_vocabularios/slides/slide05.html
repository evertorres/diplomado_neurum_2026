Definiciones Fundamentales

Elementos constitutivos de la interoperabilidad semántica.

Palabra
Un set de caracteres, incluídos signos de puntuación, delimitados por un espacio y que pueden ser encontrados en un diccionario.
Lexicón
Colección o lista de palabras con información sobre ellas.
Término
Uno o más palabras usadas como una unidad.
Concepto
Una idea, acción o cosa con un único significado.
Sinónimo
Dos términos para un mismo concepto.
Homónimos
Palabras o términos que tienen la misma grafía (se escriben igual) pero poseen un significado distinto.
Código
Una expresión en una notación formal o clasificación para un término o concepto.
